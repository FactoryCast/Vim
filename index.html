Bienvenue sur le FactoryCast consacré à Vim. Vim est un éditeur de texte puissant et fonctionnel, hérité de Vi. Mais il reste très enigmatique pour les débutants, car son fonctionnement est particulier. Avec ce FactoryCast vous connaitrez "Vim sur le bout des doigts".

Installation

  • Windows : vim.org

Les distributions

  • Spf13 : http://vim.spf13.com/
  • Vimified : http://zaiste.github.io/vimified/

Spf13

                    curl http://j.mp/spf13-vim3 -L -o - | sh

Vimified

                    curl -L https://raw.github.com/zaiste/vimified/master/install.sh | sh
